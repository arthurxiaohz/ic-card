Exporting

Export Diagram as DDL

  1. Right Click in the Diagram Editor > [Generate] > [DDL]
  2. Input a DDL filename.

Export Diagram as HTML

  1. Right Click in the Diagram Editor > [Generate] > [HTML]

  2. Input a HTML filename.

Export Diagram as Image

  1. Right Click in the Diagram Editor > [Save As Image]
  2. Input an image filename. AmaterasERD supports following image types:
    • *.png
    • *.jpg, *.jpeg
    • *.bmp
